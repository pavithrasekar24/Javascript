Ajax (Asychronous Javascript & XML)

Ajax steps

  • 0: request not initialized
  • 1: server connection established
  • 2: request received
  • 3: processing request
  • 4: request finished and response ready
  • method :GET (retrieve) , POST (save ) , PUT (update), DELETE
